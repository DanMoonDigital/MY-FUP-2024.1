## Ação

Implemente um programa que recebe um número inteiro e diga se ele é positivo ou
negativo ou nulo.

### Entrada

- Um inteiro

### Saída

- "+" se o número for positivo
- "-" se o número for negativo
- "0" se o número for nulo

## Exemplos

```txt
>>>>>>>> 0
3
========
+
<<<<<<<<

>>>>>>>> 1
-12
========
-
<<<<<<<<

>>>>>>>> 2
0
========
0
<<<<<<<<

>>>>>>>> 3
18
========
+
<<<<<<<<

>>>>>>>> 4

-
<<<<<<<<

>>>>>>>> 6
15

+
<<<<<<<<
```
